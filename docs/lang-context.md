# 🔥 Lang-Context-Probleme: Wenn der Kontext zu groß wird

## 📌 Definition
Lang-Context-Probleme entstehen, wenn der Kontext eines Modells zu groß wird.
Mögliche Probleme:
- **Token-Limit** (z.B. 128K, 1M)
- **Kontext-Verlust** (Modell vergisst frühere Informationen)
- **Performance-Einbruch** (große Contexts verlangsamen die Inferenz)
- **Halluzinationen** (Modell erfindet Fakten)
- **Tool-Call-Limit** (Agenten können nur begrenzte Tool-Calls ausführen)
